프로퍼티

		1. 프로퍼티 읽기
			- 대괄호 표기법, 마침표 표기법 두 가지
			- 대괄호 표기법의 경우 [] 안에 들어가는 값은 string 형태여야 한다.

		
		2. 프로퍼티 갱신
			- 프로퍼티로 접근하여 값을 갱신하면 된다.
			- foo.name || foo['name']  = "변경"

		
		3. 프로퍼티 동적 생성
			- 프로퍼티에 값을 할당하는 경우 프로퍼티가 존재하면 값을 갱신하고 프로퍼티가 없다면 프로퍼티 생성 후 값을 할당하는 방식을 이용

		
		4. 대괄호 표기법만 사용해아하는 경우
			- 접근하려는 프로퍼티가 에약어이거나 표현식일 경우다
				- ex ) 프로퍼티 이름 -> full-name
			- 만약에 foo.foo-name 이라고 한다면  NaN 이라는 값이 반환이 된다.(- 가 연산자기 때문이다)

      for in문과 객체 프로퍼티 출력

		for in 문을 사용하면 prop 에 프로퍼티의 이름을 담고 있다.
		var prop;
		for(prop in obj){
			console.log("prop::",prop,"  obj[prop]::::", obj[prop]);
		}

		결과 값
		prop:: name   obj[prop]:::: name123
		prop:: age    obj[prop]:::: 20
		prop:: id     obj[prop]:::: newId

		다만 console.log("prop::",prop,"  obj.prop::::", obj.prop); 으로 접근을 하면 undefined 로 반환 한다

      객체 프로퍼티 삭제

  		객체의 프로퍼티를 delete 연산자를 이용한 삭제가 가능하다.
		다만, 프로퍼티를 삭제할 수는 있지만 객체 자체를 삭제하는 것은 불가능하다.

		var fooDelete = {
		  name : "fooDeleteValue",
		  age : "fooAge"
		}

		console.log(fooDelete); -> name , age 둘 다 보유
		delete fooDelete.name;
		console.log(fooDelete); -> age 만 보유
		즉! delete라는 아이는 프로퍼티의 key 값과 value 값을 일괄적으로 삭제할 수 있는 것이다.
