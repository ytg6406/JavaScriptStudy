배열

		** 자바스크립트의 배열은 c나 java의 배열 처럼 사이즈에 대한 지정을 하지 않아도 된다.
                * 객체가 프로퍼티의 이름으로 접근을 했다면 배열의 경우 인덱스 값을 이용하여 접근을 한다.+
		
                * 배열의 요소 생성
                        - 배열의 경우 동적으로 값을 추가 할 수 있고, 순차적으로 넣을 필요 없이 아무 인덱스 값에나 추가가 가능하다.

                * 배열의 length 프로퍼티 : 모든 배열은 length 프로퍼티를 가지고 있다.
                        - length 프로퍼티는 배열의 가장 큰 인덱스에 +1 을 한 값이다.(배열의 인덱스는 0부터 length는 1부터 익숙하지 않으면 혼동을 많이 하는 부분)

                        /* 테스트 소스 */
                        var lengthArr = [];
                        console.log("배열 선언 초기 한 이후 length :::",lengthArr.length); -> 0

                        lengthArr[0] = 1;
                        lengthArr[1] = 2;
                        lengthArr[2] = 3;
                        lengthArr[99] = 100;
                        console.log("배열 선언 완료 한 이후  length :::",lengthArr.length); -> 100

                        - 이러한 배열의 length는 명시적으로 변경이 가능합니다.
                                - length의 프로퍼티를 벗어나는 인덱스에 해당 하는 값은 삭제가 이루어 진다.

                        /* length 프로퍼티 소스 */
                        var lengthChg = [0,1,2];
                        console.log("lengthChg 선언 후 length:::", lengthChg.length);

                        lengthChg.length = 6;
                        console.log("lengthChg.length 6 선언 후 배열 값:::", lengthChg); -> 기존에는 undefined x3이라고 표현이 되던 것이 신규 크롬에선 empty x 3으로 표기 되는 듯하나 확인이 필요하다.
                        console.log(lengthChg);

                        lengthChg.length = 1;
                        console.log("lengthChg.length 1 선언 후 배열 값:::", lengthChg); -> [0]

                * 배열의 표준 메서드와 length 프로퍼티
                        - 배열의 메서드들은 length 프로퍼티를 기반으로 동작한다.
                        - push 메서드는 표준 배열 메서드 이고, 배열의 마지막에 값을 넣어주는 메서드이다.

                        /* push 샘플 소스 */
                        var arrPush = ['1','2','3'];
                        arrPush.length = 4;                -> length를 변경했기 때문에 해당하는 인덱스 번호의 값은 undefined가 들어가 있다.
                        arrPush.push('4');
                        console.log("arrPush::::",arrPush); -> ["1", "2", "3", empty, "4"]
		
                * 배열과 객체
                        /* 테스트소스 */
                        var arrayTest = ["1","2","3"];
                        var objectTest = {
                                "1":"11",
                                "2":"22",
                                "3":"33"
                        }
                        console.log("arrayTest[0]:::",arrayTest[0]);        // 1
                        console.log("arrayTest[1]:::",arrayTest[1]);        // 2
                        console.log("arrayTest[2]:::",arrayTest[2]);        // 3

                        ** key 값은 "" string 타입으로 들어가야 하지만 자바스크립트 엔진에서 [] 안에 숫자형태가 들어오면 자동으로 문자로 변경해준다.
                        console.log("objectTest[1]:::",objectTest[1]);      // 11
                        console.log("objectTest[2]:::",objectTest[2]);      // 22
                        console.log("objectTest[3]:::",objectTest[3]);      // 33

                        ** array, object 로 다르지만 typeof 로 확인을 해보면 둘다 object 로 반환한다.
                        console.log("typeof arrayTest:::::",typeof arrayTest);  // object
                        console.log("typeof objectTest::::",typeof objectTest); // object
